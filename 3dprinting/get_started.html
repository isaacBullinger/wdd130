Get Started

STL

Stereolithography, or STL is the file format for a 3D file. It consists of a physical object made of thousands of triangles. This file must be converted into code that can be interpreted by the printer. The code that sends commands to the printer to make the movements, heat the plastic to the right temperature, extrude the plastic, and run the cooling fans to print the object is called gcode.

Slicer

A slicer is the software that “slices” the STL file into individual layers that the 3D printer prints. Two excellent options are Ultimaker Cura and Prusa’s Slic3r. To slice a file, drag it from your computer into the slicer and click on the slice button. Temperature is the temperature the plastic is heated to for melting it.
